Отличаются они в основном от русских звуков в следующем: 1) наряду с твёрдым <span class="text-gr">''ц''</span> имеется особый звук (фонема) — мягкое <span class="text-gr">''ць''</span>, 2) <span class="text-gr">''ч''</span> произносится твёрдо (//тш//), 3) <span class="text-gr">''в''</span> в конце слова и перед согласными произносится всегда звонко, напоминая неслоговое <span class="text-gr">''у''</span>, напр.: <span id="text-gr">кев</span> (а не <span class="text-gr">кеф</span>) //камень//, <span class="text-gr">кавксо</span> (а не <span class="text-gr">кафксо</span>) //восемь//. Для передачи эрзянских звуков в письме используется русский алфавит, причём полностью сохраняется русская система письма. ''Обозначение твёрдых и мягких согласных'' Имеется восемь пар твёрдых и мягких согласных: <span class="text-gr">''д''</span> и <span class="text-gr">''дь''</span>, <span class="text-gr">''з''</span> и <span class="text-gr">''зь''</span>, <span class="text-gr">''л''</span> и <span class="text-gr">''ль''</span>, <span class="text-gr">''н''</span> и <span class="text-gr">''нь''</span>, <span class="text-gr">''р''</span> и <span class="text-gr">''рь''</span>, <span class="text-gr">''с''</span> и <span class="text-gr">''сь''</span>, <span class="text-gr">''т''</span> и <span class="text-gr">''ть''</span>, <span class="text-gr">''ц''</span> и <span class="text-gr">''ць''</span>. Обозначение мягкости производится обычными средствами, при помощи <span class="text-gr">''ь''</span>. Если два мягких согласных из числа указанных стоят рядом, то мягкость первого не обозначается, например, пишется: <span class="text-gr">весть</span> //однажды//, хотя мягко не только <span class="text-gr">''т''</span>, но <span class="text-gr">''с''</span>, <span class="text-gr">сёксня</span> //осенью//, хотя мягко не только <span class="text-gr">''н''</span>, но и <span class="text-gr">''с''</span>. Исключением является <span class="text-gr">''л''</span>, мягкость которого всегда обозначается, напр. <span class="text-gr">кельне</span> //язычок//. ''Обозначение звонких и глухих согласных'' Имеется семь пар звонких и глухих согласных: <span class="text-gr">''б''</span> и <span class="text-gr">''п''</span>, <span class="text-gr">''г''</span> и <span class="text-gr">''к''</span>, <span class="text-gr">''д''</span> и <span class="text-gr">''т''</span>, <span class="text-gr">''дь''</span> и <span class="text-gr">''ть''</span>, <span class="text-gr">''ж''</span> и <span class="text-gr">''ш''</span>, <span class="text-gr">''з''</span> и <span class="text-gr">''с''</span>, <span class="text-gr">''зь''</span> и <span class="text-gr">''сь''</span>. Обозначение звонких и глухих согласных производится только согласно произношению. Исключение делается для одного случая: если основа имени или глагола оканчивается на звонкий согласный, то обозначение звонкости согласного выдерживается последовательно во всех формах, включая формы, где дальше следует глухой согласный, под влиянием которого приглушается предшествующий звонкий. Например, пишется: <span class="text-gr">сад</span> //сад// и <span class="text-gr">садт</span> //сады//, <span class="text-gr">куз</span> //ель// и <span class="text-gr">кузт</span> //ели//, <span class="text-gr">андан</span> //накормлю// и <span class="text-gr">андтано</span> //накормим//, <span class="text-gr">панжан</span> //открою// и <span class="text-gr">панжтано</span> //откроем//. [[<<<|grammatika]]''Имя существительное'' (лемвал) - это часть речи, которая обозначает предмет и отвечает \ на вопросы <span class="text-gr">''ки?''</span> //кто?// и <span class="text-gr">''мезе?''</span> //что?// Вопрос <span class="text-gr">''кие?''</span> //кто?// в эрзянском языке ставится только человеку: \ <span class="text-gr">ломань</span> //человек//, <span class="text-gr">цёра</span> //мужчина, сын// <span class="text-gr">ава</span> //женщина, мать//. \ Животным и всем неодушевлённым существительным ставится вопрос <span class="text-gr">''мезе?''</span> //что?//: \ <span class="text-gr">овто</span> //медведь//, <span class="text-gr">ривезь</span> //лиса//, <span class="text-gr">куз</span> //ель//, <span class="text-gr">пиземе</span> //дождь// и т. д. Существительные изменяются по падежам. Падежи в эрзянском языке выражаются с помощью окончаний (и послелогов). \ Число падежей в эрзянском языке вдвое больше, чем в русском языке. \ Кроме того, падежи составляют систему, в которой различается три типа \ склонения - основное (неопределённое), указательное (определённое) и притяжательное. Формы ''основного (неопределённого)'' склонения употребляются для обозначения предмета вообще, \ не конкретизируя и не выделяя его из ряда ему подобных. Различается 12 падежей, множественное число грамматически выражается только \ в именительном падеже (номинативе). В системе указательного склонения как в единственном, так и в \ множественном числе различается 10 падежей. В притяжательном склонении нет \ отдельного показателя множественного числа. характеризующая действия с количественной, качественной или обстоятельственной стороны. \ Основная синтаксическая функция его — быть обстоятельством. Многие наречия сохранили следы близости к именам, от которых они произошли, \ например: <span class="text-gr">маласо</span> //близко//, <span class="text-gr">малава</span> //поблизости//, <span class="text-gr">алов</span> //вниз//. По своему значению наречия делятся на следующие группы: 1. Наречия образа действия, отвечающие на вопрос \ <span class="text-gr">''кода?''</span> //как?//: <span class="text-gr">састо</span> //медленно//, <span class="text-gr">ялго</span> //пешком//, <span class="text-gr">ласте</span> //верхом//. 2. Наречия места, отвечающие на вопросы \ <span class="text-gr">''косо?''</span> //где?//, <span class="text-gr">''косто?''</span> //откуда?//, <span class="text-gr">''ков? козонь?''</span> //куда?//: \ <span class="text-gr">васоло</span> //далеко//, <span class="text-gr">маласо</span> //близко//, <span class="text-gr">тесэ</span> //здесь//, <span class="text-gr">тосо</span> //там//, <span class="text-gr">мекев</span> //обратно//. 3. Наречия времени, отвечающие на вопросы <span class="text-gr">''зярдо?''</span> //когда?//, <span class="text-gr">''зярс?''</span> //до какого времени?//: \ <span class="text-gr">исяк</span> //вчера//, <span class="text-gr">умок</span> //давно//, <span class="text-gr">курок</span> //скоро//, <span class="text-gr">ней</span> //теперь//, <span class="text-gr">валскес</span> //до утра//. 4. Наречия цели и причины, отвечающие на вопросы <span class="text-gr">''мекс?''</span> //почему?// 5. Наречия количества и меры, отвечающие на вопросы <span class="text-gr">''зяров?''</span> //на сколько?//, <span class="text-gr">''зяроксть?''</span> //сколько раз?//: \ <span class="text-gr">колмов</span> //на три части//, <span class="text-gr">кавксть</span> //дважды//. __''Образование наречий.''__ Наречия в эрзянском языке почти от всех частей речи: прилагательных, существительных, числительных, местоимений, глаголов — \ образуются при помощи специальных суффиксов. От ''качественных прилагательных'' они образуются посредством суффиксов \ <span class="text-gr">''-сто, -ста (-сте), -с, -о (-в), -кс''</span>. \ Они имеют значение образа действия или времени: \ <span class="text-gr">вадря-сто</span> //хорошо//, <span class="text-gr">ашо-сто</span> //добела//, <span class="text-gr">пси-стэ</span> //сгоряча//, <span class="text-gr">превей-стэ</span> //разумно//, но <span class="text-gr">пар-сте</span> //хорошо//, <span class="text-gr">од-с, од-ов</span> //снова//, //заново//. В наречиях, заимствованных из русского языка, употребляются \ суффиксы <span class="text-gr">''-сто, -стэ''</span> или <span class="text-gr">''-а''</span>: <span class="text-gr">дружнасто</span> (или <span class="text-gr">дружна</span>) //дружно//, <span class="text-gr">честнойстэ</span> (или <span class="text-gr">честна</span>) //честно//, <span class="text-gr">вечна</span> //вечно//, <span class="text-gr">точнасто</span> (или <span class="text-gr">точна</span>) //точно//. Многочисленная группа наречий образовалась ''из застывших падежных форм имен существительных'' \ (отложительного, местного, направительно-вносительного и переместительного падежей). Для этого служат следующие суффиксы: 1) <span class="text-gr">''-не, -нэ, -ня, -на, -да, -а, -ть''</span>, \ образующие наречия времени, отвечающие на вопрос \ <span class="text-gr">зярдо?</span> //когда?//: <span class="text-gr">кизна</span> //летом//, <span class="text-gr">тунда</span> //весной//, <span class="text-gr">чить</span> //днем//, <span class="text-gr">сёксня</span> //осенью//; 2) <span class="text-gr">''-нек, -нэк''</span>, образующие наречия образа действия. \ <span class="text-gr">''-нэк''</span> ставится только после твердых парных согласных: <span class="text-gr">колхознэк</span> //всем колхозом//, <span class="text-gr">класснэк</span> //всем классом//, <span class="text-gr">цёранек-аванек</span> //и мужчины, и женщины//; 3) <span class="text-gr">''-цек, -сек''</span>, образующие наречия образа действия: <span class="text-gr">сэрцек</span> //рядом//, <span class="text-gr">рядсек</span> //рядом//; 4) <span class="text-gr">''-кс''</span>, образующий наречия образа действия. Не нужно путать наречия на <span class="text-gr">''-кс''</span> и существительные на <span class="text-gr">''-кс''</span> в превратительном падеже. \ Наречия отвечают на вопрос <span class="text-gr">кода?</span> //как?//, а существительные — на вопросы <span class="text-gr">кикс? мезекс?</span> //в кого?//, \ //для кого?//, //во что?//, //для чего?//, //кем?//, //чем?//: \ <span class="text-gr">Мон кортан рускс, эрзякс ды немецекс.</span> //Я говорю по-русски, по-эрзянски и по-немецки//; \ <span class="text-gr">Мон сонзэ ловия эрзякс, сон жо мокша.</span> //Я его считал эрзянином, он же мокшанин//. От ''количественных числительных'' наречия образуются при помощи суффиксов <span class="text-gr">''-ксть, -нст, -в''</span> и слова <span class="text-gr">''кирда''</span> (послелог): \ <span class="text-gr">кавто — кавксть</span> //дважды//, <span class="text-gr">колмо — колмоксть</span> //трижды//, <span class="text-gr">кавтов</span> //надвое//. Суффикс <span class="text-gr">''-га''</span> образует наречия от ''глаголов'': <span class="text-gr">сонзэ марямга</span> //по его мнению//, <span class="text-gr">весень содамга</span> //общеизвестно//. ''Отместоименные'' наречия получают, используя те же суффиксы, что и в наречиях, образованных от существительных: \ <span class="text-gr">се-стэ</span> //тогда//, <span class="text-gr">то-сто</span> //оттуда//, <span class="text-gr">то-в</span> //туда//. ''Неопределенные'' наречия употребляются с частицами <span class="text-gr">''-бути, -кой, -як, -гак, -как''</span>: <span class="text-gr">''-бути''</span> всегда занимает постпозитивное положение: <span class="text-gr">косо-бути</span> //где-то//, <span class="text-gr">зярдо-бути</span> //когда-то//; <span class="text-gr">''-кой''</span> употребляется препозитивно: <span class="text-gr">кой-косо</span> //кое-где//, <span class="text-gr">кой-зярдо</span> //иногда//; <span class="text-gr">''-як, -гак, -как''</span> сливаются с наречиями и употребляются постпозитивно: <span class="text-gr">кодаяк</span> //никак//, <span class="text-gr">тесэяк</span> //и здесь//. ''Сложными'' являются наречия, которые образованы из двух слов: \ <span class="text-gr">мекев-васов</span> //взад-вперед//, <span class="text-gr">тей-тов</span> //туда-сюда//, <span class="text-gr">эль-эль</span> //еле-еле//, <span class="text-gr">цють-цють</span> //чуть-чуть//. __''Сравнительная степень качественных наречий.''__ В эрзянском языке сравнительная степень наречий образуется аналитически, с помощью слов <span class="text-gr">''ceдe''</span> //более//, <span class="text-gr">''весемеде''</span> //всех//: \ <span class="text-gr">ceдe вадрясто</span> //получше//, <span class="text-gr">весемеде парсте</span> //лучше всех//, а также путем постановки подчиненного наречию слова \ в форме отложительного падежа: <span class="text-gr">тонавтнемс ялгадо парсте</span> //учиться лучше товарища//. При наречиях стоят отрицания <span class="text-gr">''а, аволь''</span>, пишущиеся раздельно \ (отрицание <span class="text-gr">''а''</span> в нескольких словах пишется слитно: <span class="text-gr">аламо, аламос, аламне, аламодо, аламнешка</span> //немного//, \ <span class="text-gr">аютко</span> //некогда//, <span class="text-gr">аволь умок</span> //недавно//, <span class="text-gr">а курок</span> //не быстро//, <span class="text-gr">а беряньстэ</span> //неплохо//). Прилагательные в ''положительной степени'' называют качество или признак предмета без сравнения с другими предметами: <span class="text-gr">вадря ломань</span> //хороший человек//, <span class="text-gr">мазый тейтерь</span> //красивая девушка//, <span class="text-gr">кувака теле</span> //длинная зима//. ''Превосходная степень'' показывает, что тот или иной предмет превосходит остальные предметы по какому-либо признаку. Превосходная степень образуется двумя способами. ''Первый (аналитический) способ''.
